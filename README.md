# 🤖 AI Study Assistant

An AI-powered study assistant built with **Python, Streamlit, and Groq LLM** that helps students learn and understand concepts through interactive conversations.

---

## 🚀 Features

- Ask any study-related questions
- Get AI-powered explanations
- Simple chat interface
- Fast responses using Groq API
- Lightweight and easy to run

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API
- LLM (LLaMA models)

---

## 📂 Project Structure
- app.py
- ui.py
- .gitignore
- README.md 

⚙️ How to Run This Project
1. Install dependencies
pip install streamlit groq python-dotenv
2. Set up environment variables

Create a .env file in the project folder:

GROQ_API_KEY=your_api_key_here

3. Run the application
streamlit run ui.py
